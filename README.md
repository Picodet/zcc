# zcc

Zero-Config Claude Bootstrap (Linux v1).

## 用法

```bash
npx zcc install
```

第一版当前可用与预留：
1. 环境检查
2. Claude Code 安装（官方 curl / npm 兼容）
3. API/代理最小配置（保存到 `~/.zcc/config.json` 或 `~/.zcc/.env.zcc`）
4. Skills 导入插件接口（占位）
5. 飞书初始化插件接口（占位）

> npm 安装链路仅作兼容，官方推荐 `curl -fsSL https://claude.ai/install.sh | bash`。
